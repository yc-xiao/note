2.gitlab
    同git相同操作，clone需要账号密码方式或ssh方式。
    git项目内如果有子项目需要导入
        1.初始化本地子模块配置文件
        git submodule init
        2.更新项目，抓取子模块内容。
        git submodule update
    PS：https://www.cnblogs.com/codingbit/p/git-clone-with-submodule.html
